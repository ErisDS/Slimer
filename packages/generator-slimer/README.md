# Generator Slimer

## Development 

## Install

1. `git clone` this repo & `cd` into it as usual
2. Run `yarn` to install top-level dependencies.
3. Run `yarn link` to make the generator available 

## Run
- Use: `yarn dev [name]`

## Test
- `yarn lint` run just eslint
- `yarn test` run lint && tests
